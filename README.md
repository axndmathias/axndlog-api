## Projeto

 API REST com Spring Boot utilizando o MySQL, para entregas de produtos.

## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:
-  [Java 11]
-  [Spring boot]
-  [Maven]
-  [Swagger]
-  [Flyway]
-  [MySql Database]
-  [Spring Data Jpa]
-  [Postman]
-  [Lombok]
